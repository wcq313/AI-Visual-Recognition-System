# AI-Image-OCR-System
基于百度智能云的AI图文识别系统，包含**OCR通用文字识别**与**通用物体/场景识别**两大核心功能，Python+Flask前后端分离开发，界面美观、功能稳定，可直接用于课程作业与技术演示。

---

## 🎯 项目简介
本项目是一款集成式AI图文识别工具，依托百度智能云开放平台的成熟API，实现了两大核心能力：
1.  **OCR通用文字识别**：精准提取图片中的中英文、数字、符号等文字内容，支持印刷体、手写体识别
2.  **通用物体/场景识别**：智能识别图片中的物体、场景、商品等内容，返回置信度排序的识别结果
项目采用前后端分离架构，Flask作为后端服务，前端实现卡片式美观交互界面，操作简单、响应迅速，免费额度充足，适合课程作业、技术学习与二次开发。

---

## ✨ 核心功能
| 功能模块 | 功能说明 | 技术亮点 |
|----------|----------|----------|
| OCR文字识别 | 上传图片一键提取全部文字，支持多语言、多场景识别 | 百度通用文字识别API，识别准确率高、响应快 |
| 通用物体识别 | 上传图片智能识别物体/场景，返回置信度排序结果 | 百度通用物体和场景识别API，覆盖万级物体类别 |
| 前后端分离 | Flask后端提供API接口，前端独立实现交互界面 | 解耦前后端，便于维护、扩展与部署 |
| 卡片式美观界面 | 现代化UI设计，交互流畅，视觉层次清晰 | 适配课程作业展示，体验感拉满 |
| 双密钥独立管理 | OCR与图像识别使用独立API密钥，权限分离、安全稳定 | 避免密钥混用，符合开发规范 |
